FAQ

What is RRPG ?
RRPG is a platform that provide user to create a game that can interact in the real physical world
How does the RRPG provide the REAL physical gaming exprience?
Using the apps in smart phone as the player status and command device, we can connect the player status and command to the game server and interact with other players. Using NFC we connected the player and physical enviroment to the game system
It sounds similar to the Google Ingress
Ingress itself is a game where as RRPG is a platform that allow user to create their own version of game. RRPG is using NFC card as the brige og the game and the physical enviroment where Ingress is using camera to overlay the game to the physical world.
What is so interesting about RRPG?
RRPG based game provide more physical interaction between players. And the creative on the game is unlimited. It can used to create family game event or larger community game event.
